<h2 id="en-us">🇺🇸 English Version</h2>

**Nextvance** is a high-performance institutional landing page for digital creation agencies. This project showcases advanced front-end engineering using **Vanilla JavaScript**, focusing on performance, accessibility, and premium UX.

### 💎 Technical Highlights
* **Intelligent Search Bar:** Real-time DOM filtering with smooth scroll navigation.
* **Custom i18n System:** Dynamic language switching without page reloads using data-attributes.
* **Modern Performance:** Uses Intersection Observer API for optimized animations.
* **Clean Architecture:** Modular JS and CSS Variables for easy maintenance.

---

<h2 id="pt-br">🇧🇷 Versão em Português</h2>

A **Nextvance** é uma landing page institucional de alta performance para agências de criação digital. O projeto demonstra o uso de engenharia de front-end avançada com **JavaScript Puro**, focado em performance, acessibilidade e UX premium.

### 💎 Diferenciais Técnicos
* **Barra de Busca Inteligente:** Sistema de filtragem do DOM em tempo real com navegação por scroll suave.
* **Sistema de Tradução (i18n):** Troca dinâmica de idiomas via data-attributes, sem recarregar a página.
* **Performance Moderna:** Uso da Intersection Observer API para disparar animações apenas quando visíveis.
* **Arquitetura Limpa:** JS modular e Variáveis CSS para facilitar a manutenção.
